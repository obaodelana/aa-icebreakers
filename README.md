# Adullam Academy: Ice Breakers

Interactive, mission-aligned ice breaker games designed for Adullam Academy standup sessions.

## 🚀 Live Access
Play the games live at: [https://obaodelana.github.io/aa-icebreakers/](https://obaodelana.github.io/aa-icebreakers/)

## 🎮 Available Games
- **Emoji Algorithm:** A Bible-themed decryption game where students decode stories, characters, and events from emoji strings.

## 🛠 Tech Stack
- HTML5 / CSS3 (AA Brand Guided)
- Vanilla JavaScript
- GitHub Pages

## 📖 Integration
This repository is integrated as a git submodule in the main Adullam Academy project under `Docs/Playbook/Games/aa-icebreakers`.
